Hooks 列表

useCounter

计数器 Hook，提供递增、递减、重置等功能

基础用法

使用默认配置的计数器

0
直接输入数值
const { count, increment, decrement, reset, setCount, isMin, isMax } = useCounter();

带限制的计数器

设置最小值和最大值限制

5
直接输入数值
const { count, increment, decrement, reset, setCount, isMin, isMax } = useCounter({
  "min": 0,
  "max": 10,
  "initialValue": 5
});
